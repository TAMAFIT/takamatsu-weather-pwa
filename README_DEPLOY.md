# 高松天気チェッカー mock UI v6.5.3 temp pill fix

ファーストビューの気温ピルがスマホ幅で折り返していた問題を修正した版です。

## 修正内容

- ファーストビューの気温文言を短縮
  - 旧: 現在26℃　最高27℃ / 最低24℃
  - 新: 今26℃ / 高27℃ / 低24℃
- 温度ピルに `white-space: nowrap` を追加
- スマホ時の文字サイズと余白を微調整
- 画面幅が厳しい場合は省略記号で潰れ防止
