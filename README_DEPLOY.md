# 高松天気チェッカー mock UI v4

評価・天気・雨リスクの役割を分離した版です。

## 主な修正

- ファーストビュー大見出しを短縮
  - 海日和 / 行ける / 条件付 / 見送り
- ファーストビュー説明文を短く整理
- 1時間カードのアイコンは「実際の天気」で決定
  - 快晴/晴れ: 太陽
  - 一部曇り: 太陽+雲
  - 曇り: 雲
  - 雨/雷: 雨雲
- 1時間カードの色は「評価」で決定
  - 最高: ゴールド
  - 良い: ミント/水色
  - 微妙: 薄オレンジ
  - 弱い: 青灰
- 雨確率はアイコンに混ぜず、右側ピルで表示
- 「良い」など評価文字がカードに隠れにくいようカラム幅を調整
- 雲 / 日照 / UV / 雨 の数値は省略させない方針を維持
- API取得、判定ロジック、9:00〜17:00評価、背景WebP、夜背景は今日だけ、は維持
