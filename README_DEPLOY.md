# 高松天気チェッカー mock UI v6.5.6 sun strength

1時間カードの「日照60分」表示を、直感的な「日差し 強い / あり / 弱め / なし」に変更した版です。

## 修正内容

### 1. 日照分数をそのまま見せない

旧表示:

- 日照 60分

新表示:

- 日差し 強い
- 日差し あり
- 日差し 弱め
- 日差し なし

### 2. 日差しの強弱を総合判定

単純に `sunshine_duration` だけではなく、以下を合わせて判定します。

- 日照時間
- UV
- 短波放射
- 雲量
- 雨天コード / 降水量

### 3. 数値データは内部では維持

ロジックや診断ログでは日照分数を引き続き保持します。
画面上の1時間カードだけ、利用者向けに自然な文言へ変換しています。
