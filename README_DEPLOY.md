# 高松天気チェッカー mock UI v5

UIの意味づけを整理した版です。

## 変更点

- 時間別ラベルを `最高 / 良い / 普通 / 弱い` に変更
  - `微妙` は廃止
- アイコンは「天気コード + 雲量 + 日照」で決定
  - 快晴/晴れ: 太陽
  - 晴れだが雲あり: 太陽+雲
  - 雲多め: 雲+太陽
  - 曇り: 雲
  - 実際の雨/雨天コード: 雨
- 雨確率だけでは雨アイコンにしない
- 小見出しも `快晴 / 晴れ / 薄雲 / 雲多め / 曇り / 雨` に変換
- カード色は評価で決定
  - 最高: ゴールド
  - 良い: ミント
  - 普通: 白〜薄ベージュ
  - 弱い: 青灰
- 数値ピルは時間カード下段に逃がし、見切れにくく変更
- API取得、9:00〜17:00判定、背景WebP、夜背景は今日だけ、は維持
