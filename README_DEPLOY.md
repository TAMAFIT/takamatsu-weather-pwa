# 高松天気チェッカー mock UI v6.5.2 temp plan A

プランAとして、UIを崩さずに気温情報を追加した版です。

## 修正内容

### 1. ファーストビューに気温サマリーを追加

判定文の下に小さく、

- 現在気温
- 最高気温
- 最低気温

を表示します。

例:

- 現在27℃　最高31℃ / 最低24℃

今日以外を選択している場合は、

- 最高31℃ / 最低24℃

として表示します。

### 2. 1時間ごとのカードに気温を追加

各カードの中央テキスト側に、

- 良い
- 薄雲
- 27℃

のように気温を追加しました。

右側のピル群は維持します。

- 雲
- 日照
- UV
- 雨

### 3. 週間カード温度修正は維持

v6.5.1の `0/0℃` 修正も維持しています。
