# 高松天気チェッカー mock UI v6.4.2 polish

v6.4.1の取得正常化後、表示・診断ログを整えた版です。

## 修正内容

### 1. 夜間のアメダス日照表示を自然化

夜に今日を見ている場合、直近日照0hをそのまま「日照なし」と出すと誤解しやすいため、

- 実測日照: 夜間

と表示するようにしました。

判定コメントも、

- 実測日照は夜間のため参考外

に変更します。

### 2. 安定度「初回」を「記録開始」に統一

表示揺れをなくすため、

- 初回
- 記録開始

が混在しないようにしました。

### 3. 自前安定度は維持

更新ごとの通常予報履歴をlocalStorageに保存し、次回以降に

- 高
- 中
- 低

へ変わります。

### 4. 診断ログのrecentFetches重複を整理

fetch記録が同じURLで二重に出ていたため、短時間の同一URL・同一statusはまとめて `count` を付けるようにしました。
