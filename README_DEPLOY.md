# 高松天気チェッカー mock UI v6.5 auto verify

手動入力に頼らず、保存した予測をあとから衛星日射・アメダス実測で自動照合する版です。

## 追加内容

### 1. 予測スナップショットを自動保存

更新時に、今日または未来日の判定をlocalStorageへ保存します。

保存するもの:

- 判定: 海日和 / 行ける / 条件付 / 見送り
- スコア
- 良い時間数
- 日照合計
- 最大UV
- 雨リスク
- 風/波
- おすすめ時間
- 通常予報/JMA/衛星/海況の要約

今日の18時以降は、その日の予測を上書きしません。

### 2. 自動外れ判定

保存済みの予測について、対象日が終わったあとに自動照合します。

照合材料:

- 衛星日射
- 衛星推定日照
- アメダス日照
- アメダス雨量

ラベル:

- 的中
- 的中寄り
- やや外れ
- 外れ
- 保留

### 3. 自動外れ判定カードを追加

画面下部に「自動外れ判定」を追加しました。
手動入力なしで、予測と実測のズレを蓄積します。

## 注意

完全な体感評価ではありません。
「海に行って満足だったか」「実際に焼けたか」は人間の体感が入るため、自動判定はあくまで

- 日射
- 日照
- 雨

から見た自動照合です。
